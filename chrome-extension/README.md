# YouTube Transcript Extractor — Chrome Extension

Extract full transcripts from any YouTube video with one click.

## Features

- Extract transcript from any YouTube video that has captions
- Click any line to jump to that moment in the video
- Search through the transcript in real-time
- Copy the full transcript to clipboard (with timestamps)
- Auto-highlights the current segment as the video plays

## Installation (Load Unpacked)

1. Open Chrome and go to `chrome://extensions`
2. Enable **Developer mode** (toggle in the top-right corner)
3. Click **Load unpacked**
4. Select the `chrome-extension` folder from this project
5. The extension icon will appear in your Chrome toolbar

## How to Use

1. Go to any YouTube video (e.g. `youtube.com/watch?v=...`)
2. Click the extension icon in your toolbar
3. Click **Extract Transcript**
4. The transcript loads — click any line to seek to that moment
5. Use the search bar to find specific words or phrases
6. Click the copy icon to copy the full transcript with timestamps

## Notes

- Only works on videos that have captions/subtitles available
- Prefers English captions; falls back to the first available track
- The extension requests no personal data — it only reads the page you're on
